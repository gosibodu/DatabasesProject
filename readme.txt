README

Control the cursor with the arrow keys. use PgUp and PgDn to go up and down on the z axis

To set up for yourself:
  Use mySql and PHP5
  Set your webserver to the www folder
  create a user in mysql named 'compsci' with the password 'databases'
  create a database called 'cs2300'
  grant the user compsci all privileges on cs2300
  run databaseinitialization.sql
